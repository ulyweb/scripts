Frequently Asked Questions

How do I run the PowerShell script?
Open PowerShell and run the script using `.\WinBackup2Box.ps1`. You may need to set execution policy with `Set-ExecutionPolicy RemoteSigned`.
What is Box and how is my data secured?
Box is a cloud storage service. Your data is encrypted both in transit and at rest. This tool helps automate local backups to Box.
Can I use these tools on macOS?
Yes! The “Backup Your Data to Box.app” is a native macOS application for automated local backup to your Box folder.
