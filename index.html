Frequently Asked Questions

How do I run the PowerShell script?
Right-click the file and choose "Run with PowerShell" or open PowerShell and run the script using `.\WinBackup2Box.ps1`.
What is Box and how is my data secured?
Box is a cloud storage service. Your data is encrypted both in transit and at rest. This tool simply automates local backups to your Box folder.
Can I use these tools on macOS?
Yes! The “Backup Your Data to Box.app” is a native macOS application you can extract and use on any modern Mac.
